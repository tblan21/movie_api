This is a web application which will contain a list of movies. Server-side programming will allow the user to sign up and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get a list of movies /movies GET None None A JSON object holding data about all movies
Get data about a single movie /movies/:title GET :title None A JSON object holding data about a single movie
Get data about a genre by genre name /movies/genre/:genreName GET :genreName None A JSON object holding data about a specific movie genre
Get data about a director by the director's name /movies/directors/:directorName GET :directorName None A JSON object holding data about a specific director
Allows new users to register /users POST None A JSON object holding data about the new user, structured like: { "Name": "String" } A JSON object holding data about the user that was added, structures like: { "Name": "String" }
Allows users to add a movie to their list of favorites /users/:Name/movies/:MovieID POST :id :movieTitle None A message showing that the movie was successfully added
Allows users to remove a movie from their list of favorites /users/:Name/:favoriteMovie DELETE :id :movieTitle None A message showing that the movie was successfully removed
Allows existing users to deregister /users/:Name DELETE :id None A message showing that the user was removed
